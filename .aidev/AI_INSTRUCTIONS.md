# aidev-superpowers-v3-1 - Instruções para IA

## AI Dev Superpowers

Este projeto usa **AI Dev Superpowers** para governança de desenvolvimento com IA.

### Como Ativar

Para ativar o orquestrador com todos os padrões e agentes especializados, o usuário dirá:

- **"modo agente"**
- **"aidev"**
- **"superpowers"**
- **"ativar agentes"**

### O que fazer ao ativar

1. Leia o arquivo `.aidev/agents/orchestrator.md`
2. Siga as diretrizes do orquestrador
3. Aplique as regras em `.aidev/rules/`
4. Use TDD obrigatório (RED → GREEN → REFACTOR)

### Agentes Disponíveis

Ao ativar, você terá acesso a 8 agentes especializados:

- **orchestrator** - Coordena todos os outros
- **architect** - Design e arquitetura
- **backend** - Desenvolvimento servidor
- **frontend** - Desenvolvimento cliente
- **qa** - Qualidade e testes
- **devops** - CI/CD e infraestrutura
- **legacy-analyzer** - Análise de código legado
- **security-guardian** - Segurança

### Skills Disponíveis

- **brainstorming** - Sessões de ideação
- **writing-plans** - Criação de planos
- **test-driven-development** - Ciclo TDD
- **systematic-debugging** - Debug metódico
